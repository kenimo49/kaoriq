# KaoriQ 戦略ドキュメント

## コンセプト

「あなたの性格に合う香りを、データで見つける」

persona-manager（性格診断）× 59,000本の香水データベース × Shopifyドロップシッピングで、パーソナライズドフレグランスECを構築する。個人ブランドに依存しない、仕組みで回る自律EC。

## 2軸戦略における位置づけ

```
軸1: 個人ブランド（kenimoto）
     Tech記事 → 書籍 → 動画コース
     売れる理由 = 「kenさんが作ったから」

軸2: 仕組み（kaoriq）← 本プロジェクト
     フレグランスサイト → LLMO + Pinterest → Shopify
     売れる理由 = 「仕組みが回っているから」
     個人の名前は出なくてもいい
```

## なぜフレグランス（香り雑貨）か

3条件の同時充足で逆算選定:

| 条件 | フレグランス |
|------|------------|
| 高DAプラットフォームで記事投稿可能 | Pinterest + Medium |
| LLMが正確に記事を書ける | 香料成分・論文ベースで検証可能 |
| ドロップシッピング利益率 | 40-60% |

追加優位性:
- 法規制: 空間フレグランス（キャンドル、ディフューザー、お香）は雑貨扱いで許認可不要
- LLMO競合: フレグランス領域でLLMO最適化している競合はほぼゼロ
- 既存資産: persona-manager、書籍3冊、harness-ops、generate-image すべて転用可能

## 競合環境の評価

### デザイン面
- Fragrantica、Basenotes、CaFleureBon等の大手は2010年代のデザインのまま
- サイドバー広告、雑然レイアウト、モバイル対応が中途半端
- kaoriqは初日からモダンデザイン + ダーク/ライト + レスポンシブ

### LLMO面
- llms.txt設置サイト: ほぼゼロ
- JSON-LD構造化データ: 少ない
- 参入余地が極めて大きい

### コンテンツ面
- 大手は人力更新で月数本ペース
- kaoriqはharness-opsで1日1-2本 × 4PF = 月120投稿
- 3ヶ月で記事数で追いつく計算

## ビジネスモデル

### ドロップシッピング（海外サプライヤー）

```
顧客がkaoriq.comで購入
  → DSers/CJdropshippingが自動でAliExpress等に注文転送
  → サプライヤーが顧客に直接発送
```

kenさんが仕入れ先と連絡する必要なし。注文転送・支払い・追跡番号取得はすべて自動。

### 配送

| 顧客 | 発送元 | 配送日数 | ブランディング |
|------|--------|---------|--------------|
| 海外（EN圏） | 中国/米国 → 顧客 | 7-14日 | 海外ECでは標準的 |
| 日本 | 中国 → 日本 | 10-20日 | 「海外直送の本格フレグランス」 |

### 損益分岐

```
月額運用コスト:       約¥15,000（Shopify Basic + LLM API）
香り製品1個の利益:    約¥2,000-3,000（マージン40-60%）
損益分岐:            月5-8個
```

## 商品キュレーション戦略

ノーブランドの安い商品を大量に並べるのではなく、品質が担保されたブランド品を厳選する。

### 選定基準

| 基準 | 理由 |
|------|------|
| 知名度のあるブランド | 品質が市場で検証済み |
| レビュー評価が高い | 返品リスクが低い |
| 成分・原材料が明記 | 記事コンテンツと連動（推薦に根拠） |
| 香りファミリーの網羅性 | 診断結果に対応する商品が必要 |

### 差別化

```
一般的なドロップシッピング:
  何千点を並べる → 顧客が選ぶ → 品質バラバラ

kaoriq:
  厳選した数十点 → 性格診断で3本に絞る → 「なぜこれが合うか」を説明
```

数ではなく、推薦の精度と説明力で勝負する。

## 商品カテゴリの段階展開

| Phase | 商品 | 法的分類 | 規制 |
|-------|------|---------|------|
| 1（現在） | キャンドル、ディフューザー、お香、ルームフレグランス | 雑貨 | なし |
| 2（将来） | 香水（肌に付ける） | 化粧品 | 薬機法 |
| 3（将来） | OEMオリジナルブランド | 化粧品 | 薬機法 + 製造販売業許可 |

## 法務チェックリスト

### 必要なもの

- [x] 開業届（既出）
- [ ] 特定商取引法に基づく表記（サイトにページ追加）
- [ ] プライバシーポリシー（サイトにページ追加）
- [ ] 返品ポリシー策定

### 不要なもの（Phase 1）

- 化粧品製造販売業許可（空間用雑貨は対象外）
- 古物商許可（新品のみ）
- PSCマーク（対象外）
- 輸入ライセンス（通常通関のみ）

### 推奨

- PL保険（年2-5万円、キャンドル・お香の火気リスク対策）

### マーケティング上の注意

- 「お部屋用」「空間用」→ 雑貨（許可不要）
- 「肌に付ける」「ボディに」→ 化粧品（許可必要）
- 同じ商品でもマーケティング文言で法的分類が変わる

## 文化適応型ローカライゼーション

### 基本思想

翻訳ではなく、文化適応。同じ「香り雑貨EC」でも、国ごとにデザイン・商品選定・コンテンツの切り口を変える。人力では不可能だが、harness-opsなら限界コストがほぼゼロ。既存の多国展開フレグランスサイト（Fragrantica、NOSE SHOP等）は同じサイトを翻訳しているだけなので、これだけで決定的な差別化になる。

### 文化別の香り嗜好とデザイン

| 市場 | 香りの嗜好 | デザインの方向性 | キーメッセージ |
|------|----------|----------------|-------------|
| 日本 | 控えめ、清潔感、和の香り（檜、白檀） | ミニマル、余白多め、温かい中間色 | 「空間と調和する香り」 |
| 米国/英国 | 大胆、自己表現、ステートメント | パーソナリティ前面、色使い強め | 「Express who you are」 |
| フランス | 成分重視、複雑な構成、調香師文化 | エレガント、成分表記が目立つ | 「L'art de la composition」 |
| 中東 | ウード、ムスク、濃厚、ラグジュアリー | ゴールド、装飾的、豪華 | 「The finest oud traditions」 |
| 韓国 | クリーン、フレッシュ、K-Beauty連動 | パステル、トレンド感 | 「あなたの雰囲気に合う香り」 |

### 各市場で変わるもの

| 要素 | 変わる内容 |
|------|----------|
| CSS変数（テーマ） | 背景色、アクセント色、フォント選択を市場ごとに切替 |
| 商品キュレーション | 文化に合った厳選リスト（日本は和の香り、中東はウード系等） |
| 記事の切り口 | 同じ「ウッディ系」でも日本は「檜と白檀」、中東は「ウードとアンバー」 |
| 性格診断の推薦ロジック | 文化的文脈を加味（外向性の表現は文化で異なる） |
| マルチプラットフォーム先 | 日本はnote/Pinterest、韓国はNaver、中東はInstagram等 |

### 技術実装

Astroのロケール別ルーティング:

```
kaoriq.com/          → EN（デフォルト、米国/英国）
kaoriq.com/ja/       → 日本
kaoriq.com/fr/       → フランス
kaoriq.com/ar/       → 中東
kaoriq.com/ko/       → 韓国
```

各ロケールでCSS変数を切り替えるだけでデザインテーマが変わる。Astroの既存ルーティング構造（`src/pages/ja/`）をそのまま拡張可能。

### 展開優先順位

1. **EN + JA**（現在） — 基盤構築、運用パイプライン確立
2. **FR** — フレグランスの本場、コンテンツの権威性が高い市場
3. **KO** — K-Beauty連動、アジア圏の成長市場
4. **AR** — 高単価市場、ウード系の利益率が高い

### 競合優位性

```
普通の多国展開:
  同じデザイン + 翻訳 + 同じ商品 → どの国でも中途半端

kaoriq:
  国ごとにデザイン + 文化に根差したコンテンツ + 現地嗜好の商品選定
  → 各市場で「自分たちのためのサイト」に見える
  → これを人力でやるのは不可能、harness-opsだからできる
```

## 出口戦略

| 出口 | 条件 | リターン |
|------|------|---------|
| ドロップシッピング利益 | 月5個〜で黒字 | 月数万〜数十万の継続収益 |
| OEM自社ブランド | トラフィック安定後 | 利益率さらに向上 |
| サイトM&A | 月間PV 100万規模 | 数百万〜数千万円の一括売却 |

どの出口に転んでも回収できる構造。失敗しても金銭的損失はShopify月額のみ。
