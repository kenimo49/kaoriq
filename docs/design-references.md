# デザインリファレンス

kaoriq.com のデザイン方針と参考サイトをまとめる。

## デザイン方針: Warm Intellectual

「温かみのある知性」をコンセプトに、フレグランス業界の上質感とデータ駆動の信頼感を両立させる。

### カラーパレット

| 用途 | ダークモード | ライトモード | 意図 |
|------|------------|------------|------|
| 背景 | `#1A1714` 温かいチャコール | `#FAF7F2` パーチメント | アース系で香りの温かみ |
| カード | `#2A2520` | `#FFFFFF` | コンテンツ領域の区別 |
| テキスト主 | `#E8E0D6` | `#2C1810` エスプレッソ | ブラウン系で柔らかい印象 |
| テキスト副 | `#C4B8A8` | `#4A3828` | 本文の読みやすさ |
| アクセント | `#C4972A` アンバー/ゴールド | `#B8860B` | フレグランス業界の定番色 |
| リンク | `#D4A96A` | `#9A7B4F` ディープアンバー | 紫→アンバーに変更 |
| セパレータ | `#3D3630` | `#E0D8CE` | 温かいグレー |

### 避けるべき色

- バイオレット/紫: Tech/開発者ブログの印象が強い
- 冷たいネイビー: フレグランス業界と合わない
- ホットピンク: ジェンダーニュートラルなポジショニングに反する
- ピュアブラック背景: ファッション寄りすぎ、温かみがなくなる

### フォント構成

| 要素 | フォント | 種類 | 効果 |
|------|---------|------|------|
| 見出し（h1/h2/h3）、ロゴ | Cormorant Garamond | セリフ | フレグランスの上質感・感性 |
| 本文 | DM Sans | サンセリフ | データの信頼感・読みやすさ |
| データラベル（将来） | DM Mono | モノスペース | 分析・推薦スコアの知的さ |

セリフ見出し + サンセリフ本文の組み合わせは、Maison Louis Marie、Diptyque等のフレグランスブランドで実績のあるパターン。

## ロケール別デザインリファレンス

「高級感」の表現は市場ごとに真逆。フランスは「引き算」、中東は「装飾」、韓国は「モノクローム」、米国は「パーソナリティ」。1つのデザインで全市場に出すと必ずどこかでミスマッチが起きる。kaoriqはCSS変数のロケール切替で各市場に最適化する。

### 実装方法

既存の `data-theme` （dark/light）に加え、`data-locale` 属性でCSS変数を切替:

```html
<html data-theme="light" data-locale="kr">
```

```css
[data-locale="kr"] {
  --bg-primary: #FFFFFF;
  --accent-primary: transparent;
  font-family: 'Pretendard', 'Inter', sans-serif;
}
[data-locale="kr"] h1, [data-locale="kr"] h2, [data-locale="kr"] h3 {
  font-family: 'Pretendard', 'Inter', sans-serif;  /* セリフを外す */
}
[data-locale="ar"] {
  --accent-primary: #D4A83A;
  direction: rtl;
  font-family: 'Cairo', 'DM Sans', sans-serif;
}
[data-locale="fr"] {
  --accent-primary: #B8860B;  /* アンティークゴールド */
}
```

`data-theme` × `data-locale` の2軸で、テーマ切替と市場別デザインが同時に動く。
